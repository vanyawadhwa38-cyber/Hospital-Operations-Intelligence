# Hospital Operations Intelligence
## Overview

Hospital Operations Intelligence is an end-to-end healthcare analytics project built using Python, SQL, and Power BI. The project analyzes 55K+ patient records to uncover operational insights related to hospital performance, patient demographics, admissions, billing, medical conditions, and resource utilization.

---

## Tech Stack

- Python
- Pandas
- NumPy
- SQL
- Power BI
- DAX

---

## Project Workflow

1. Data Cleaning & Preprocessing using Python
2. Feature Engineering
3. SQL Analysis
4. Power BI Dashboard Development
5. Interactive Business Insights

---

## Key Features

- Data preprocessing and feature engineering
- SQL-based healthcare analysis
- 15+ DAX measures
- Interactive Power BI dashboards
- KPI cards
- Dynamic slicers
- Revenue and billing analysis
- Patient segmentation
- Hospital performance analysis
